1. FEATURES:
- Current weather information including temperature, humidity, wind speed, and weather description
- Five-day weather forecase with daily temperature and weather description
- Line graph representation of temperatyre variations over the forecasted days
- Display of current local time in the specified city's timezone
- User-friendly graphical interface for easy interaction

2. PROJECT DESCRIPTION:
This application is a Python program that provides users with the ability to quickly access accurate weather information and forecasts for a specific city.
The app fetches weather data from the OpenWeatherMap API and presents it in a user-friendly graphical interface.

3. HOW TO INSTALL & RUN PROJECT:
- Clone repository to local machine
- Navigate to project directory
- Install required libraries and dependencies 
     - Python 3.x
     - PySimpleGUI
     - Matplotlib
     - requests
     - geopy
     - timezonefinder
- Run application 
     - The app will open a graphical interface window
     - Enter the name of a city in the input field and click the "Enter" button
     - The app will fetch weather data and display current conditions, a five-day forecast, and a temperature graph
     - The local time of the specified city will be displayed
